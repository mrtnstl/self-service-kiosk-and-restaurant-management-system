# self-service-kiosk-and-restaurant-management-system

the restaurant used as an example is a ficticiuos sushi place with minimalist, pastel visuals

## Use Cases (self service kiosk)

- as a user, i want to select if i'm dining at the restaurant or i want my order to go, so...
- as a user, i want to browse the available dishes separated by categories, so i can find the desired food items more quickly
- as a user, i want to see my order summed up before checkout, so i can make modifications or continue browsing if necessary
